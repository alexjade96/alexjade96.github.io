## Intro

A personal learning repo for working through ML and data engineering concepts hands-on. Each topic lives in its own isolated environment with its own `.venv` and `requirements.txt`.

## LearnPyTorch

The main focus of the repo. Working through deep learning fundamentals using PyTorch, following the [learnpytorch.io](https://www.learnpytorch.io/) curriculum. Practical exercises include transfer learning experiments with EfficientNet (B0, B2) and ResNet18 applied to image classification — including early experiments using League of Legends champion splash art as a dataset, which later grew into its own standalone project ([League of LLM](/projects/league-of-llm)).

## LearnPySpark

Early exercises following the [sparkbyexamples.com](https://sparkbyexamples.com/pyspark-tutorial/) PySpark tutorial, covering distributed data processing fundamentals.
